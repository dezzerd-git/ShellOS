# ShellOS
ShellOS - (for now) this is a visual Windows shell made entirely with the help of AI, since I don't know how to program. Initially, the project was written in PowerShell, but since version 0.8 it has been written entirely in the C# programming language. In the future, it is planned to make it cross-platform and make our own Linux distribution of the ShellOS version. The project itself originated from the idea of "what if" and now has many versions. I do not plan to freeze the project, but you understand me, if I do not release versions for a long time, this does not mean that the project is dead or frozen. I am a student, I do not have much time, and the AI itself is often stupid, which is why progress it slows down or even has to roll back. I hope you enjoy my ShellOS project and I am also ready to accept any constructive criticism (ShellOS is completely made with the help of AI, since I cannot program myself)
launch instructions: to launch a version older than ShellOS v0.7, you need to download the project and put its unpacked zip file where it is convenient for you to right-click on the file and select "run with PowerShell" in the drop-down window. Wait a bit and ShellOS will start, after which you can use it, but remember the PowerShell window or the window The command line (cmd) cannot be closed, otherwise ShellOS will also close. 
to run versions of ShellOS v 0.7 and v0.7.1, you will need to download the file and put it where it is convenient for you to open the folder with ShellOS, find the bat file named "ShellOS", click on it 2 times and wait a little bit after which ShellOS opens (the bat file does not work if it is located outside the root folder of ShellOS) but remember just like in older versions, you cannot close the PowerShell window and the command line (cmd), otherwise ShellOS will close with it after completing work in ShellOS, close it through the cross in the upper-right corner, go to the PowerShell window or command line (cmd) from which it opened ShellOS and press ENTER, then the file will close by itself
to launch the version of ShellOS v0.8 and later, you will need to download the zip file from ShellOS, unpack it where it will be convenient for you to go to the ShellOS folder and run the ShellOS file (.exe), then wait a bit and ShellOS will start, BUT the .exe file cannot be removed from the folder in versions otherwise ShellOS won't start.
